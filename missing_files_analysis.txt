ROUTE RESOLUTION LOGIC (from layouts/app_layout.php)
=====================================================
When a user accesses home.php?route={route_name}, the system:
  1. Checks: components/dashboards/{route_name}.php  (dashboard file)
  2. Checks: pages/{route_name}.php                  (page file)
  3. If NEITHER exists -> Shows "Page not found" error

Note: Only root-level files in pages/ are resolvable. Files in
subdirectories (e.g., pages/staff/admin_staff.php) are NOT directly
route-accessible; they are only included by root-level page files.
